# Format specifier

```
    { [field_name] [!modifier] [:specification] }
```
- field_name:      name (.attribute_name | [index] ) *
- modifier:        r (internal view) | s (human view)
```python
'Unit: {players[0]}'.format(players=['1', '2', '3'])    # 'Unit: 1'
'Unit: {players[0]!s}'.format(players=['1', '2', '3'])  # 'Unit: 1'
'Unit: {players[0]!r}'.format(players=['1', '2', '3'])  # "Unit: '1'"
```
- specification:    
```
    [ [fill] align ] [sign] [#] [0] [width] [,] [.precision] [type]


+------------+-----------------------------------------------------------+
| fill:      | any symbol expect **'{'** and **'}'**                     |
+------------+-----------------------------------------------------------+
```

- align
```
+---+------------------------------------------------------------------------------+
| < | symbols of fill place on right (alignment by left side) (it's default value) |
+---+------------------------------------------------------------------------------+
| > | alignment by right side                                                      |
+---+------------------------------------------------------------------------------+
| = | symbols of fill after sign but before digits (for number value only)         |
+---+------------------------------------------------------------------------------+
| ^ | alignment by center                                                          |
+---+------------------------------------------------------------------------------+
```

- sign
```
+-------------+---------------------------------------------------------+
| +           | the sign use for all numbers                            |
+-------------+---------------------------------------------------------+
| -           | sign show for negative numbers only                     |
+-------------+---------------------------------------------------------+
| " " (space) | '-' for negative values and 'space' for positive values |
+-------------+---------------------------------------------------------+

```

```
+------------+-----------------------------------------------------------+
| width:     | integer value which define the width of the field         |
+------------+-----------------------------------------------------------+
| precision: | same, but define precision of the value                   |
+------------+-----------------------------------------------------------+
```

- type
```
+---------+-----------------------------------------------------------------------------------+
| b       |                                                                                   |
+---------+-----------------------------------------------------------------------------------+
| c       | symbol                                                                            |
+---------+-----------------------------------------------------------------------------------+
| d, i, u | integer value                                                                     |
+---------+-----------------------------------------------------------------------------------+
| e       | float number in science format (letter 'e' in lower case)                         |
+---------+-----------------------------------------------------------------------------------+
| E       | float number in science format (letter 'e' in upper case)                         |
+---------+-----------------------------------------------------------------------------------+
| f, F    | float number general format                                                       |
+---------+-----------------------------------------------------------------------------------+
| g       |                                                                                   |
+---------+-----------------------------------------------------------------------------------+
| G       |                                                                                   |
+---------+-----------------------------------------------------------------------------------+
| o       | number in octal number system                                                     |
+---------+-----------------------------------------------------------------------------------+
| s       | string                                                                            |
+---------+-----------------------------------------------------------------------------------+
| x       | number in hex number system (letters in lower case)                               |
+---------+-----------------------------------------------------------------------------------+
| X       | number in hex number system (letters in upper case)                               |
+---------+-----------------------------------------------------------------------------------+
| %       | float number in percentage format (the number multiply by 100 and add symbol '%') |
+---------+-----------------------------------------------------------------------------------+
```

### examples
```python
coord = (3, 5)
'X: {0[0]};  Y: {0[1]}'.format(coord)           # 'X: 3;  Y: 5'
```
```python
"repr() shows quotes: {!r}; str() doesn't: {!s}".format('test1', 'test2')
# "repr() shows quotes: 'test1'; str() doesn't: test2"
```
```python
'{:<30}'.format('left aligned')                 # 'left aligned                  '
'{:>30}'.format('right aligned')                # '                 right aligned'
'{:^30}'.format('centered')                     # '           centered           '
'{:*^30}'.format(' centered ')                  # '********** centered **********'
```
```python
'{:+f}; {:+f}'.format(3.14, -3.14)              # '+3.140000; -3.140000'
'{: f}; {: f}'.format(3.14, -3.14)              # ' 3.140000; -3.140000'
'{:-f}; {:-f}'.format(3.14, -3.14)              # '3.140000; -3.140000'
```
```python
"int: {0:d};  hex: {0:x};  oct: {0:o};  bin: {0:b}".format(37)
# 'int: 37;  hex: 25;  oct: 45;  bin: 100101'

'int: {0:d};  hex: {0:#x};  oct: {0:#o};  bin: {0:#b}'.format(37)
# 'int: 37;  hex: 0x25;  oct: 0o45;  bin: 0b100101'
```
```python
points = 19.5
total = 22
'Correct answers: {:.2%}'.format(points/total)     # 'Correct answers: 88.64%'
```
